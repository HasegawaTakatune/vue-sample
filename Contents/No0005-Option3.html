トップ

Life cycle/ Hook

ライフサイクル、イベントハンドラ、メモだけ残す。
beforeCreate インスタンスが作成される直前に呼ばれます。
created インスタンスが作成された直後に呼ばれます。
beforeMount render によってマウントされる直前に呼ばれます。
mounted render によってマウントされた直後に呼ばれます。
beforeUpdate インスタンスのデータが更新される直前に呼ばれます。
updated インスタンスのデータが更新された直後に呼ばれます。
activated コンポーネントが活性化する時に呼ばれます。
deactivated コンポーネントが非活性化する時に呼ばれます。
beforeDestroy インスタンスが破棄される直前に呼ばれます。
destroyed インスタンスが破棄された直後に呼ばれます。
errorCaptured エラーがキャプチャされた時に呼ばれます。

Directives

ディレクティブを自作できる。ここでは、フォントサイズを変えているだけだがイベントフックで各タイミング別に動作させられそう。
    <div id="app">
        <div v-myfont_8>Hello!!</div>
        <div v-myfont_12>Hello!!</div>
        <div v-myfont_16>Hello!!</div>
        <div v-myfont_20>Hello!!</div>
        <div v-myfont_24>Hello!!</div>
    </div>
    <script>
    var app = new Vue({
        el: '#app',
        directives: {
            myfont_8: {
                inserted: function (el) {
                    el.style.fontSize = '8pt';
                },
            },
            myfont_12: {
                inserted: function (el) {
                    el.style.fontSize = '12pt';
                },
            },
            myfont_16: {
                inserted: function (el) {
                    el.style.fontSize = '16pt';
                },
            },
            myfont_20: {
                inserted: function (el) {
                    el.style.fontSize = '20pt';
                },
            },
            myfont_24: {
                inserted: function (el) {
                    el.style.fontSize = '24pt';
                },
            },
        },
    });
    </script>

Filters

{{ message1 | toUpper }}
{{ message1 | toLower }}
もとの値からフィルタ掛けをする。カナのフィルタ掛けなどにも使えそう。
    <div id="app">
        <div>{{ message | toUpper }}</div>
        <div>{{ message | toLower }}</div>
    </div>
    <script>
    var app = new Vue({
        el: '#app',
        data: {
            message: 'Hello!!',
        },
        filters: {
            toUpper: function (value) {
                return value.toUpperCase();
            },
            toLower: function (value) {
                return value.toLowerCase();
            },
        },
    });
    </script>

Mixins


ライブラリみたいに取り込んで利用することができるみたい、実装する際のお約束として見ておけばいい感じ。
    <div id="app">
        <input type="button" @click="hello" value="Hello">
    </div>
    <script>
    var MyMixin = {
        methods: {
            hello: function () {
                alert("Hello!");
            },
        },
    };
    var app = new Vue({
        el: '#app',
        mixins: [MyMixin],
    });
    </script>
